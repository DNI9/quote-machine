## Setup / Installation

Clone this repo and cd into it

```sh
git clone https://github.com/DNI9/quote-machine.git
```

Install dependencies

```sh
yarn
```

## Run locally

Develop

```sh
yarn dev
```

To build

```sh
yarn build
```
